BioShooter

Description

This game follows a doctor inside a patient's body tasked to eliminate all viruses. A flying robot as a character companion follows the doctor and provides emotional and gameplay support. The game consists of three stages each separated by a door. Players have to find the key to access the next stage. The third level is intentionally balanced to be frustrating. This game was developed as part of my Master Thesis. In this project, I was responsible for:

Responsibilities

  • Implementing and designing 3 frustrating levels inside Unity Editor
  • Implementing 5+ enemy AI systems using C# scripts
  • Implementing AI companion system using C# scripts
  • Implementing dialogs and event systems using C# scripts and Unity UI systm
  • Designing 5+ enemy variations and move sets using C# scripts
  • Implementing companion audio system using Ai voice generators (Replica Studio)
  • Implementing various sound effects such as enemy shooting, hit sound, etc
  • Balancing game resources using Excel and Fibonacci series
  • Optimizing for browser using Unity Profiler and texture compression techniques, improving load times up to 60%
