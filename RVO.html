Crowd Simulation RVO

This project enhances RVO (Reciprocal Velocity Obstacles) algorithm specifically proposes solutions for the problem with a crowd dense static agents. The proposed solution consists of several steps including adding static agents to navmesh and the navmesh was changed to dynamic. Additionally, the static agents were made bumpable. Meaning that whenever agents had no path to take to, the static agents would adjust their position a bit so a path is made. Moreover, the performance of three distinguished algorithm using our method was investigated namely, DCM (Detour Crowd Manger), RVO and agents with no avoidance algorithm. This project was done in Unreal Engine. In this project, I was responsible for:

  • Implementing levels and scenarios
