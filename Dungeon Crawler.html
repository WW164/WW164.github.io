Dungeon Crawler

Description

The game is a 2D rogue-like dungeon crawler following an astronaut, who strives to survive and escape the spaceship that is infested with alien monsters. The goal of the player is to defeat the final boss and escape. In the process of finding the boss and escaping, the player will come across two different types of enemies, health packs, treasure objects, and multiple rooms. In this project, I was responsible for:

Responsibilities

  • Implementing player movement
  • Implementing player shooting
  • Implementing UI
  • Implementing room transitions
  • Implementing game flow
  • Implementing triggers for data collection
  • Implementing levels
