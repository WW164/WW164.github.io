House Paint

During my master's degree, we developed House Paint, a 2D mobile game on Android platform based on a similar hypercasual game called House Paint as part of our Mobile Interaction course. Aim of this project was to test three different input methods, swipe, UI buttons, and tilt. In this project, we implemented Game Analytics to gather data from our participants and draw conclusion on each input method. I was responsible for:

  • Brainstorming and designing the initial concept of the game
  • Implementing Game Analytics SDK
  • Defining and fully implementing custom events
  • Working with Game Analytics Dashboards and extracting the data
