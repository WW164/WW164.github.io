Save the Princess

Save the Princess is a runner hypercasual game in which players can merge soldiers, defeat the end level boss and save the princess. This game successfully passed the CPI test and currently I'm working on its 1st iteration on Retention phase. In this project, I was responsible for:

  • Designing the core loop and initial concept
  • Conceptualizing the concept
  • Level design
  • Organizing and prioritizing tasks
  • Plan roadmap and milestones
  • Play testing and bug report
